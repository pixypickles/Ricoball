# Ricoball MVP v13

## ボタン配置

三角配置を維持しつつ、左から見て A / B / C になるよう変更しました。

- 左下: A
- 上中央: B
- 右下: C

Fold表示でも同じ並びです。

## ターゲット

- ターゲットを床から浮かせました
- ゴロや低すぎるボールはターゲットに当たりません
- 低いボールはターゲットの下を通過します
- ターゲットに支柱と影を追加しました
- ターゲット全体へ固定の傾きを追加しました
- 中心命中でも完全にはまっすぐ反射しません
- 命中位置による湾曲反射も維持しています

## 維持している操作

- A: 強シュート / スライディング / ボレー
- B: ループシュート / ショルダーチャージ / ヘディング
- C: ジャンプ
- AIは首位チームのターゲットを優先します
