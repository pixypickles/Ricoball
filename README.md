# Ricoball MVP v12

## 操作

### A
- ボールあり: 強シュート
- ボールなし: スライディングタックル
- 空中でボールが近い: ボレーシュート

### B
- ボールあり: ループシュート
- ボールなし: ショルダーチャージ
- 空中でボールが近い: ヘディング

### C
- ジャンプ

## 変更点

- カーブシュートを完全廃止
- ボタン配置を左から A / B / C に変更
- ループシュートを追加
- ショルダーチャージを、前方へ約1キャラ分踏み込んですぐ元位置へ戻る動作へ変更
- ショルダーチャージ命中時は相手を大きく吹き飛ばす
- AIは現在トップのチームのターゲットを優先
- 同点首位の場合は近いターゲットを選択
- 高いジャンプ、空中移動、Fold対応UI、湾曲ターゲットを維持

`index.html`だけで動作します。
