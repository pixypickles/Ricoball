# Ricoball MVP v3

4チーム・各1人で戦う、リバウンド型ターゲットアリーナのブラウザ試作です。

## 実装済み

- 4チーム各1人（プレイヤー1人＋AI 3人）
- 侵入禁止エリアなし
- 丸みのある四隅
- 各辺中央の楕円形ターゲット
- 外側2点、中4点、中心6点
- 被弾チームは得点の半分を減点
- ターゲットから上向きにリバウンド
- 円形アナログスティック
- A：ストレートシュート／足を出す／ボレー
- B：カーブシュート／スライディング／ヘディング
- C：ジャンプ／ジャンプトラップ
- A・B長押しチャージ
- 簡易カーブ自動補正
- 浮き球・重力・バウンド
- 100秒マッチ

## 操作

### スマートフォン
左スティックで移動。A/Bは長押しして離すとチャージ技、Cはジャンプです。

### PC
- 移動：WASD または矢印
- A：J
- B：K
- C：L

## 起動

`index.html` を開くか、このフォルダで以下を実行します。

```bash
python3 -m http.server 8000
```

その後 `http://localhost:8000` を開いてください。

## GitHub Pages

リポジトリのルートへ全ファイルを置き、Settings → Pages でメインブランチのルートを公開元に設定します。
