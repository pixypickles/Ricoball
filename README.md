# Rebound Royale — Mobile Web MVP

巨大なターゲットとリバウンドを使う、4チーム制のサッカー風バトルロイヤルです。

## MVPのルール

- 正方形コートの4辺に、各チームの巨大ターゲットがあります。
- ターゲットに当てたチームは得点します。
- 当てられたターゲットのチームは、得点の約1/4を失います。
- ターゲットの得点は外側から **1 / 5 / 10 / 20点**。
- ボールはターゲットに当たっても止まらず、そのまま跳ね返ります。
- ターゲット前の半透明エリアはプレイヤー侵入禁止です。ボールだけ入れます。
- 自分のターゲットに当てると小さなペナルティがあります。
- 90秒終了時に最高得点のチームが勝利します。
- 現在は1人＋AI 3人です。

## 操作

### スマートフォン

- 左下スティック：移動
- 右下 `KICK`：キック

### PC

- `WASD` または矢印：移動
- `Space`：キック

## 起動方法

依存ライブラリやビルド作業はありません。

### いちばん簡単

`index.html` をブラウザで開きます。

ブラウザのセキュリティ設定によってローカルファイルが制限される場合は、フォルダ内で簡易サーバーを起動してください。

```bash
python3 -m http.server 8000
```

その後、ブラウザで `http://localhost:8000` を開きます。

## GitHub Pagesへの公開

1. このフォルダの中身をGitHubリポジトリへ追加
2. GitHubのリポジトリ設定から **Pages** を開く
3. デプロイ元としてメインブランチのルートを指定
4. 発行されたURLをスマートフォンで開く
