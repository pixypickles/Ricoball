# Ricoball MVP v10

## コート変更

- 四辺の中央が内側へ反ったコートへ変更
- コーナー側は広く、ターゲット付近は狭い形状
- 壁の傾きに応じた法線反射を実装
- 壁へまっすぐ当てても、位置によって斜めに跳ね返る
- ターゲットを反った辺の中央へ配置

## シュート改善

- シュート可能距離を拡大
- 足元付近のボールを蹴る瞬間に収める補助を追加
- 少し浮いたボールも地上シュート可能
- シュート速度を少し上昇
- クールタイムを短縮
- 成功時にSHOT / CURVE SHOTを表示
- A/Bを離した時にシュート

Version 9のジャンプ、空中移動、ボレー、Fold対応UIも維持しています。
