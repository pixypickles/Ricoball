# Rebound Royale (Revised Rules)

## コアコンセプト
4人（4チーム1人推奨）で戦う、リバウンド型ターゲットアリーナ。

## コート
- 正方形
- 四隅は丸くしてボールが滞留しにくい設計
- 四辺中央に少し前傾した巨大ターゲット

## ターゲット
- 同心円ターゲット
- 外側：2点
- 中間：4点
- 中心：6点
- ボールは上方向へ少し跳ね返る
- 当てたチーム：得点
- 当てられたチーム：獲得点の半分を減点

## 操作
左スティック：移動

ボール保持時
- A：ストレートシュート（長押しチャージ）
- B：カーブシュート（長押しチャージ）
- C：ジャンプ

ボール非保持時
- A：足を出して奪う
- B：スライディングタックル
- C：ジャンプ

浮き球付近
- A：ボレー
- B：ヘディング
- C：ジャンプトラップ
