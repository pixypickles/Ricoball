# Ricoball MVP v5

## Version 5の主な変更

### キャラクター表示
- キャラクター全体を進行方向へ回転させない方式へ変更
- 頭は常に画面上向き
- 腕と脚の動きで走行やアクションを表現
- 足元の小さな矢印で進行・照準方向を表示
- キック、ボレー、ヘディング、ジャンプ、ショルダーチャージ、スライディング、被弾姿勢を表示

### ドリブル
- 低速の地上ボールへ近づくと足元へ軽く吸い付く
- 完全固定ではなく、キャラクターの少し前を追従する
- 歩くだけで止まったボールを運べる
- ドリブル中は足元に点線サークルを表示
- ショルダーチャージやスライディングを受けるとボールがこぼれる
- シュート時にドリブル状態を解除

### 操作
- 左スティック / WASD / 矢印：移動
- A / J：ストレートシュート。ボールがない場合はショルダーチャージ
- B / K：カーブシュート。ボールがない場合はスライディング
- C / L：ジャンプ。浮き球付近ではトラップ
- 浮き球付近でA：ボレー
- 浮き球付近でB：ヘディング

## 起動

`index.html` をブラウザで開いてください。

ローカルサーバーを使う場合：

```bash
python3 -m http.server 8000
```

その後、`http://localhost:8000` を開きます。
