# Ricoball MVP v4

プレイフィールとアクションの見やすさを重点的に作り直した版です。

## 主な変更

- 丸い駒ではなく、頭・胴体・腕・脚のある簡易キャラクター
- 歩行モーション
- キックモーション
- ボレーの蹴り上げモーション
- ショルダーチャージの前傾姿勢
- スライディング姿勢と土煙
- ジャンプ動作
- ヘディング動作
- 被弾時の転倒姿勢
- AIに役割を与え、全員がボールへ群がりにくく調整
- Aの非保持時アクションをショルダーチャージへ変更
- Bの非保持時アクションを強いスライディングへ変更
- スライディング命中時の吹き飛ばし・硬直
- ボール速度を上昇
- 地面と壁でのバウンドを強化
- ターゲット命中時の上方向リバウンドを大幅強化
- 6点ゾーンは特に高く跳ねる
- 画面揺れと短いヒットストップ

## 操作

### スマートフォン
- 左スティック：移動
- A：ボール付近ではストレートキック、非保持時はショルダーチャージ、浮き球ではボレー
- B：ボール付近ではカーブキック、非保持時はスライディング、浮き球ではヘディング
- C：ジャンプ、浮き球付近では減速トラップ

### PC
- 移動：WASD または矢印
- A：J
- B：K
- C：L

## 起動

`index.html` を開くか、フォルダ内で次を実行してください。

```bash
python3 -m http.server 8000
```

その後、ブラウザで `http://localhost:8000` を開きます。
