# Ricoball MVP v11

## コート

- コート形状を元の丸角スクエアへ戻しました
- コートの壁は通常のまっすぐな反射です
- 的の表面だけをコート側へ湾曲させました
- 的の中心以外へ当たると、湾曲面の法線に沿って斜めに反射します
- 的の見た目も平面の楕円から、内側へ膨らむプレートへ変更しました

## シュート

チャージ操作を廃止しました。

- A：押した瞬間に強いシュート
- B：押した瞬間に曲がるシュート
- 空中A：強いボレー
- 空中B：曲がるボレー
- ボールが少し足元から離れていても、蹴る瞬間に収めてシュートします

ボールがない場合は従来通り：
- A：ショルダーチャージ
- B：スライディング

Version 9のFold対応UI、高いジャンプ、空中移動、短いスライディングも維持しています。
